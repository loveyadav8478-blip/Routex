# RouteX 🚦

AI-Powered Traffic Intelligence Platform

RouteX predicts traffic conditions using Machine Learning,
provides real-time alerts through WebSockets,
and offers AI-powered route assistance.

## Features

- AI Traffic Assistant
- Traffic Prediction Engine
- Real-Time WebSocket Alerts
- JWT Authentication
- Analytics Dashboard
- Historical Traffic Insights

## Architecture

React + Vite
        ↓
Spring Boot
        ↓
PostgreSQL + Redis
        ↓
FastAPI ML Service
